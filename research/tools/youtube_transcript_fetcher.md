# YouTube Transcript Collection Approach

## Purpose

This file documents the planned technical approach for collecting YouTube transcripts for the research project.

## Tool Options Considered

### 1. Supadata API
Supadata can be used to fetch YouTube transcripts through an API endpoint. This is useful when collecting transcript data from multiple YouTube videos in a structured way.

### 2. Free transcript methods
Alternative free methods include:
- YouTube transcript extraction tools
- Browser-based transcript copying
- Python packages such as `youtube-transcript-api`

## Planned Workflow

1. Identify relevant YouTube videos from selected experts.
2. Collect video URLs.
3. Fetch or manually extract transcripts.
4. Save transcripts under:

`/research/youtube-transcripts/`

5. Summarize each transcript into:
- main ideas
- tactical insights
- tools mentioned
- possible playbook ideas

## Current Status

Initial expert research has been completed using public articles, expert profiles, interviews, and content hubs. YouTube transcript collection is prepared as the next iteration.
